SpeedTest component allows you to test the network speed of your server.

It shows you the download, upload, ping and jitter speed by using a default button.

Testing the upload, ping and jitter the component will generate the testcases by itselfs.

For testing the download speed you will need a
uncompresseable payload file deployed on your server with a size of 1MB, 2MB, 5MB or 10MB.

Recommed is using a 5MB file to test the speed.
